## Clone and enter the repository
```
https://github.com/MarcinKadziolka/nerf-gui.git
cd nerf-gui
```
## Create virtual environment and activate it
```
python3 -m venv .venv
source .venv/bin/activate
```
## Install required packages
```
pip install -r requirements.txt
```
## Run the program
```
python3 nerf_gui.py
```
